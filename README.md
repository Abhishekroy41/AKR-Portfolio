# AKR Portfolio

A high-performance, modern, and fully responsive personal portfolio designed for a Data Analyst.

## Features
- **Modern UI/UX**: Dark-themed aesthetic featuring cyberpunk, neon-glowing accents perfectly tailored to a tech analytics professional.
- **Dynamic Animations**: Smooth scroll, typewriter effects, floating isometric 3D illustrations, and interactive hover states powered by CSS and Framer Motion.
- **Custom Graphics**: Incorporates sophisticated glowing data-dashboard illustrations and seamless transparent blending.
- **Responsive Design**: Flawlessly adapts across all screen sizes (mobile, tablet, desktop) without broken layouts or large white spaces.
- **Functional Contact Form**: Integrated with Web3Forms to receive emails directly from portfolio visitors.

## Tech Stack
- **React.js**
- **Vite**
- **Framer Motion**
- **Vanilla CSS**
- **Lucide Icons**

## Local Installation
1. Clone the repository: `git clone https://github.com/Abhishekroy41/AKR-Portfolio.git`
2. Navigate into the project: `cd frontend`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
